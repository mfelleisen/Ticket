## Milestone 8

### Release to Students

- `ForStudents`

### Files and directories for test fest 

- `xtest` is the testing script 
- `xref` is the referee-testing script 
- `Tests/` is the directory of "integration tests"

### Running tests

In this directory, run 

```
$ ./xtest Tests/ ./xref
```

or 

```
$ ./xtest ForStudents/ ./xref
```
